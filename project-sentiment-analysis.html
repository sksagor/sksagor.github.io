Technical Implementation

Backend Architecture

  • FastAPI framework for high-performance async API endpoints
  • PostgreSQL for storing analysis results and user data
  • Redis for caching frequently analyzed content and rate limiting
  • Celery for background task processing of large datasets
  • Docker containerization for easy deployment

NLP Models Used

  • HuggingFace BERT (bert-base-uncased) for transformer-based analysis
  • VADER (Valence Aware Dictionary) for social media text
  • TextBlob for general purpose sentiment detection
  • Custom ensemble model combining all three for improved accuracy
